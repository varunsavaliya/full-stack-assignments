# ToDo App

This project is a simple blog website built with [React](https://reactjs.org/) and [Material-UI](https://mui.com/). It allows users to view todos and add new todos.

## Run Locally

Clone the project

```bash
  git clone url
```

Go to the project directory

```bash
  cd directory_name
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
